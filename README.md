🎯 Number Guessing Game (Java)

📌 Project Overview

The Number Guessing Game is a simple Java console application that generates a random number within a specified range (1–100) and challenges the user to guess it. The program provides hints after each attempt, informing the player whether the guess is too high or too low, until the correct number is found.

This project is designed to strengthen fundamental Java programming concepts such as loops, conditionals, user input handling, and random number generation.


---

🚀 Features

Generate a random number between 1 and 100

Accept user guesses through the console

Provide feedback for incorrect guesses

"Too High!"

"Too Low!"


Continue until the correct number is guessed

Display a success message when the player wins


Optional Enhancements

Limit the number of attempts

Add multiple game rounds

Implement a scoring system

Display the total number of attempts taken
